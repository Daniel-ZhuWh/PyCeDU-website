Structure flow

Student Response Monitor

Teachers could publish checkpoint questions during lecture time and track students’ answer statuses. They could remind unanswered students and display the top performance student list after the response time.

Performance Comparison

Performance comparison offers a visualized 1-D scatter plot that shows the average performance and how this class performed compared with the other classes this teacher teaches.

Personalized Broadcast

Personalized broadcast includes three functions: screen sharing, resource sharing and message sending. Resource Sharing: Teachers could manage the resource library and send unit-specific resources (videos, tutorials, and slides) to specific groups of students.

Summary

Teachers will receive an auto-generated short summary immediately after the whole checkpoint question session. The system also recommends some targeted materials to support teacher involvement.

Student End: Dual Mode Practice System

Structure flow

Mr.Pangolin

Mr. Pangolin is an animated pedagogical character that can interact with students as tutors. As a companion, it can support students’ emotional states using empathy and building relationships with them.

Dual Practice Mode

We designed two practice modes for different personas. For top students who want to explore with less scaffolding, they can choose the classic practice mode. For students who are clueless or don’t know where to start, they can choose the step mode that breaks problems into chunks. The chunks follow a logical order and move toward a clear goal. Indicating how the steps fit into a hierarchy of goals and subgoals (e.g., by labeling some steps as key subgoals) in the step mode appears to aid learning.

Worked Example

A step-by-step solution to solved similar problems. It will release automatically and remind students to check when their console keeps empty. Students are told to study the example to prepare for upcoming problem solving.

Personalized Diagnosis

A code lab to help students know their code quality immediately based on test cases and AST matching. System will also offer elaborated feedback to guide future revisions to students.

Locked Advanced Step

An extended step for students who’ve finished the required part. Students will receive an invitation to unlock the advanced step once they pass the normal practice no matter in which mode.

System-assigned Learning Materials

When students are waiting for teachers’ help, they are assigned some resources for their independent learning. The are able to read some resources and try to think on their own to solve problems they encounter.

Teacher End: Real-time Analytical Dashboard

Structure flow

Requesting help list

Teachers could see a dynamic list that is updated instantly as soon as changes occur to see who asks for help. Based on this list, teachers could get a sense of the student’s practice progress, number of handins, their current progress, and give their code feedback if needed.

Challenging Test Cases

System will synthesize the top several most challenging test cases and display them to teachers. They are able to see the error rate, students who failed the test and relative reasons. With this information, teachers can either send messages to struggling students or explain the difficult points to the whole class.

Class Practice monitor

Student Activity Monitor

Teachers could see each student’s current detailed learning status, choose the sort order by different priorities, view their codes, and remind students to change modes if needed.

Class Performance Quick View

By clicking the summary button on the student information page, teachers could see the latest class performance overview of all students in a single view. It’s a time-saving way to monitor class performance quickly.

Code Feedback

Teachers are able to check individual student’s code submissions, send private message to code lines, run codes in an embeded editor, and show codes to all the class via screen sharing.

Structure flow

Teacher End: Auto-generated Performance Summary

Classroom Performance Summary

Class performance summary provides an informative but easy-to-read report for teachers to quickly see trends and activity for their entire class time, including attendance, time usage, lecture overview, practice overview, skill mastery level and highlighted students. Armed with this knowledge, teachers can make more informed class adjustments.

Student Performance Summary

One-line summary is a list in the practice summary page that displays each student’s in-class performance based on their practice journey, text matching/ test score/ and code submission history. Teachers are allowed to sort the list by different metrics, star students, print and share this list.

Unfinished Tasks

Unchecked Challenging Test Cases

It summarizes the test cases that students get a low pass rate but the teacher missed or doesn’t have a chance to check in the class time. Teachers are expected to add these common errors to their plan for the next class and explain to students before the new lecture.

Students Requesting Help

It records students who raised hands in their practice time but didn’t receive teacher’s help. With this still waiting for the help list, teachers are able to make after-class appointments with students and encourage them to solve their in-class problems.

Provided Support

It documented all the support provided by the teacher or the system in the class time. Support shown here includes personal help, messages and materials have been sent.This feature enables the teacher to check the material content in detail, students who received help, and download the detailed history if needed.

Star Symbol

A spotlight feature that allows teachers to star students who need special attention and recognize them easily in the class time.

Teacher End: Student Individual Report

Practice Journey

Inspired by Spring Idea 4, we put an outline of the student practice journey in the class summary report and a more detailed one in the student individual report to show their own practice process.

Customized Report Sharing

A report automatically compiled by system that can customize the content to download and share. What’s more, teachers are able to click specific student one-line reports and code examples, downloading to local or sharing the file.

Class Final Report

Customized Final Grading

Based on stated rubric at the beginning of the semester, the teacher is able to adjust extra marks, compose the final grade, download and share to others.
